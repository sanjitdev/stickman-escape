## File Mapping

Place downloaded files as follows:

```
public/assets/
  sprites/
    player.png          ← New Platformer Pack character spritesheet
    enemy.png           ← New Platformer Pack enemy spritesheet
    tiles.png           ← Pixel Platformer tilemap_packed.png
    coin.png            ← New Platformer Pack collectible
    key.png             ← New Platformer Pack key
    door-closed.png     ← New Platformer Pack door
    door-open.png       ← New Platformer Pack door (open frame)
    platform-moving.png ← New Platformer Pack moving platform tile
    platform-falling.png ← New Platformer Pack crumbling tile
    portal.png          ← New Platformer Pack exit portal
    checkpoint.png      ← New Platformer Pack checkpoint flag
    bg-forest.png       ← New Platformer Pack background (forest)
    bg-cave.png         ← New Platformer Pack background (cave)
    bg-fortress.png     ← New Platformer Pack background (fortress)
  audio/
    music-forest.ogg    ← Music Jingles (upbeat, loopable)
    music-cave.ogg      ← Music Jingles (mysterious)
    music-fortress.ogg  ← Music Jingles (intense)
    sfx-jump.ogg        ← RPG Audio
    sfx-double-jump.ogg ← RPG Audio
    sfx-land.ogg        ← RPG Audio
    sfx-dash.ogg        ← RPG Audio
    sfx-coin.ogg        ← RPG Audio
    sfx-key.ogg         ← Interface Sounds
    sfx-death.ogg       ← RPG Audio
    sfx-checkpoint.ogg  ← Interface Sounds
    sfx-door.ogg        ← Interface Sounds
    sfx-stomp.ogg       ← RPG Audio
    sfx-portal.ogg      ← Interface Sounds
    sfx-click.ogg       ← Interface Sounds
```
